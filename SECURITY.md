# Security Policy

## Supported Versions

Currently, only the latest version of VoiceFlow is supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of this project seriously. If you find a security vulnerability, please do NOT report it via a public issue.

Instead, please send an email to the maintainers or use GitHub's private reporting feature if available. We will acknowledge your report within 48 hours and provide a timeline for a fix.
