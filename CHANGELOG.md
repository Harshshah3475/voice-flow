# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-12

### Added
- Initial release of VoiceFlow.
- **Hold-to-Talk** dictation model.
- Real-time voice-to-text using Deepgram Nova-2.
- **Auto-Type** functionality into active windows.
- Floating minimalist widget with waveform animations.
- **History** system for tracking past transcriptions.
- **Theme Support** (Light/Dark/System).
- Configurable global hotkeys.
- BYOK (Bring Your Own Key) model for security.
